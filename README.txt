This project is part of full stack nano degree program from udacity, it's an application that provides varity of categories along with it's own items as well as provide user authentication using third party, the authorized users have the ability to create, edit, and delete their own items.

In order to run this project you need to fork the FSND-Project2 to your machine, Git Bash(for windows users), python, vagrant, virtual machine and any text edior.(which is provided at the end with all required software)

After installing all required software open Git Bash and do the following:
1- cd to FSND-Project2/vagrandt 
2- then run vagrant up command
3- vagrant ssh
4- cd to catalog directory by running cd /vagrant/FSND-Project2
5- run python categoryProject.py 
6- go to your browser and type localhost:5000/categories.

now you can use my application to create, edit or delete items, but first you have to login to access these features.
