This project is part of full stack nano degree program from udacity, it's about catalog to enable the user to create new items for any category.
the categoryItems DB contain three tables Users, categories, and items. this project enabling the user to create new itme, edit or delete an exicting one.

in order to run this project you need to install Linux virtual machine and vagrant.(which is provided at the end with all required software)
After installation you have to run these commad in git bash( if you are using windows): vagrant up => to bring the VM online then run vagrant ssh command 
and inside the directory that contains the categoryProject.py and the categoryItem DB, first run the DB file befor runing python file (python categoryProject.py).
